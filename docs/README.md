# AFC Service Design

> Service design and planning for Apostolic Faith Church digital platform modernization.

## What is this?

The [Apostolic Faith Church](https://apostolicfaith.org/) (headquartered in Portland, Oregon) serves 3,000+ churches across 60+ countries. The current website runs on Webflow CMS, requiring extensive manual work for content publishing, multilingual translation (9 languages), and audio production.

This site documents the design and strategy for building an **AFC Platform** — an internal content management system that automates translation, audio generation, and publishing.
